Depuis quelques jours mon blog est visible sur les mobiles. Il adapte un thème particulier quand un téléphone ou un PDA navigue sur le site.

J'ai fait une petite capture pour que vous voyez l'aspect obtenu.

Avec la multiplication des téléphones nouvelles générations, tel que l'iphone, de plus en plus de sites web proposent une version mobile (facebook, youtube, google, twitter, ...)

Ça m'est déjà arrivé d'avoir besoin de consulter un site web en ayant pas le net, et donc en passant par le téléphone. Mais comme l'affichage n'étais pas adapté et la page beaucoup trop lourde, la navigation était impossible.

Rendre son blog compatible

Ceci est réservé aux blogs de type Wordpress.

Il y a un plugin appelé MobilePress à installer.

Après vous pouvez choisir un titre pour le blog, une description du blog (si vous ne mettez rien il garde les réglages par défaut de votre blog. Utile si vous voulez que le titre et la description soit differente pour celui qui utilise un mobile). Vous avez aussi la possiblité d'obliger le blog à prendre l'apparence pour mobile pour tout le monde.

Et enfin vous pouvez changer le thème :)
