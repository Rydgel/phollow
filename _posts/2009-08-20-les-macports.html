Les macports sont aux Macs ce que AUR est à Arch Linux : Un système pour télécharger les sources de vos programmes, les compiler puis les installer. Ce système d'installation vient tout droit des *BSDs. Tout se passe en ligne de commande, ça rappellera à certains apt-get install et cie. Les macports contiennent environ 6000 logiciels dont la plupart se retrouve sous Linux.

Installation

Rendez-vous sur cette page et télécharger l'image .dmg. L'installation est assez traditionnelle.

Utilisation

Et hop j'installe ce superbe jeu nethack :)))

Les commandes de bases sont les suivantes :

  • port list  : pour lister les ports disponibles
  • port search : pour chercher un port
  • port install : pour installer un port
  • port uninstall : pour en désinstaller un.
