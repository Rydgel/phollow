Je vais aujourd'hui ouvrir une nouvelle catégorie sur le blog : bout de codes. Des billets courts avec des morceaux de codes qui font une chose et bien.

Dans ce billet nous allons voir un peu de CSS 3 afin de réaliser des boutons qui auront le style des applications Mac (Cocoa).

Pour arriver au résultat, il nous faut principalement 3 fonctions CSS 3 :

  • border-radius qui va s'occuper d'arrondir les angles
  • box-shadow pour les ombres
  • linear-gradient pour les dégradés
Fonctionne sur les derniers Firefox, Safari, et Chrome (et tout ce qui est à base de Webkit ou Gecko). Malheureusement Opera ne supporte pas encore les dégradés. Je ne parlerai même pas de cette daube d'Internet Explorer.
