Hurl est un service web open-source. L'idée est de proposer une version facilement utilisable de cURL sous forme d'un site web. Et donc de pouvoir tester des API super rapidement. Et le meilleur dans tout ça c'est que le code source du projet est disponible pour tout le monde.

Hurl a été crée par Leah Culver et Chris Wanstrath à l'occasion du 2009 Rails Rumble où le programme gagna le prix de l'application la plus complète.

Hurl s'utilise les doigts dans le nez, on rentre l'URL en choisissant les options qui vont bien. On peut rajouter des headers, une authentification etc. Et on reçoit directement le résultat dans la page, avec un lien unique que l'on peut partager.

Une application que je trouve bien utile. Les liens :
