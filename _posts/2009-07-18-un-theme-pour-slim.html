Slim est un manager de connexion graphique comme GDM ou encore KDM. Il est intéressant pour les personnes voulant une interface de connexion graphique sans pour autant dépendre de Gnome, Kde etc. Slim possède d'ailleurs des thèmes assez jolis qui ne sont pas difficile à faire.

Installation

sudo pacman -S slim slim-themes

Ensuite vous devez rajouter le démon slim dans votre /etc/rc.conf pour qu'il soit lancé au démarrage de l'ordinateur.

Réglages

Slim utilise votre .xinitrc pour savoir quel environnement lancer. Par exemple si vous mettez exec openbox-session dans votre .xinitrc alors ce sera Openbox qui sera lancé après votre login. Vous avez aussi la possibilité de mettre plusieurs environnements et d'en choisir un quand slim est lancé, plus d'infos ici.

La configuration globale du logiciel se fait dans /etc/slim.conf

Mon thème

C'est très minimaliste je l'accorde, mais moi j'adore. Et peut-être que d'autre aussi :)

Les thèmes se situent dans /usr/share/slim/themes, et le choix du thème ce fait dans le fichier de configuration slim.conf :

current_theme archlinux
